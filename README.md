# BigScreen -Simple for EmulationStation Desktop Edition

BigScreen -Simple theme set for ES-DE that supports most systems (wip).

The following options are included:

2 variants:

- Textlist with videos
- Textlist without videos

3 color schemes:

- Black
- Gold
- Grey

4 aspect ratios:

- 16:9
- 16:10
- 4:3
- 21:9
